## 📖 Описание проекта

Этот проект представляет собой комплексный лендинг и промо-страницу для фитнес-эксперта. Главная задача — воссоздать предоставленный дизайн с высокой точностью, не используя конструкторы сайтов или тяжелые JS-фреймворки. Код написан на чистом HTML, CSS и нативном JavaScript, что гарантирует чистоту, семантику и легкую поддержку.

В процессе разработки применялся современный итеративный подход «вайбкодинг» с использованием ИИ в качестве помощника для генерации базовой структуры, после чего проводилась тщательная ручная отладка и доводка для обеспечения безупречного выравнивания, точного соответствия цветов и идеальной кросс-браузерной адаптации.

## 🛠️ Используемые технологии

- **HTML5** – Семантическая, доступная разметка.
- **CSS3** – Flexbox и CSS Grid для гибкого построения макетов, `object-fit` для адаптации изображений, CSS-переменные (custom properties).
- **Vanilla JavaScript** – Нативная работа с DOM для реализации интерактивных элементов (без сторонних библиотек).
- **Git & GitHub Pages** – Система контроля версий и бесплатный статический хостинг.
- **VS Code** – Среда разработки.

## ✨ Ключевые особенности и UX

- **📱 Полная адаптивность:** Оптимизирован для настольных компьютеров, планшетов и смартфонов. Каждый блок плавно подстраивается под любую ширину экрана.
- **🖱️ Интерактивные карусели:** Разработаны кастомные слайдеры для отзывов и фото «до/после». Поддерживают перетаскивание мышкой (drag), клик по кнопкам для прокрутки и стилизованные скроллбары.
- **🚀 Плавная навигация и таймеры:** Реализован плавный скролл по якорным ссылкам внутри страницы, а также рабочий таймер обратного отсчета для создания эффекта срочности.
- **📦 Динамические аккордеоны:** Разделы с подробным описанием программ («Как устроено» / «Инвентарь») плавно раскрываются и сворачиваются при клике (чистый CSS/JS).
- **🖼️ Модальное окно с формой:** Кастомный всплывающий попап с функциональной лид-формой, включая переключатель согласия на маркетинговые рассылки (точная копия оригинала).
- **🎨 Управление ассетами:** Тщательно продуманная файловая структура со сквозной нумерацией изображений, исключающая дубликаты и упрощающая поддержку кода.

## ⚙️ Процесс разработки и итерации

Использование ИИ позволило ускорить создание начального макета, однако значительный объем ручного труда был вложен в итеративную отладку и исправление багов на основе постоянной обратной связи по дизайну. Ключевые решенные задачи:

1.  Исправление выравнивания Hero-блока (на десктопе фото строго справа, на мобильных — корректный сброс) с помощью Flexbox.
2.  Корректировка контрастности и CSS-переменных для точного попадания в фирменные цвета бренда.
3.  Написание кастомного JS-скрипта для перетаскивания каруселей, чтобы отказаться от сторонних библиотек.
4.  Точная настройка отступов и использования `object-fit: contain` в карточках товаров, чтобы изображения отображались полностью без обрезания, строго по оригинальному дизайну.

## 🚀 Как запустить локально

1. Склонируйте репозиторий на свой компьютер.
2. Откройте файл `index.html` в любом современном веб-браузере.
3. *(Опционально)* Чтобы открыть страницу с заказом, перейдите на файл `promo.html`.
